# NotifAi

![NotifAi Logo](app/src/main/ic_launcher-playstore.png)

A smart notification assistant for Android.

## Features

* **Smart Filtering:** Automatically filters and prioritizes your notifications.
* **Customizable Rules:** Create your own rules to manage notifications.
* **AI-Powered Suggestions:** Get smart suggestions for new rules based on your notification
  history.
* **Notification Log:** Keep a log of all your notifications.
